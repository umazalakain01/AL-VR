Welcome to ALVR

This wizard will guide you to the basic setup of ALVR
Firewall
To communicate with the headset, some firewall rules need to be set. This requires administrator rights.
Driver
The ALVR driver needs to be registred with SteamVR
Tracking
How should the tracking of the controller be handled. f you plan on playing fast pased games like Beatsaber, choose medium or fast. For slower games like Skyrim leave it on normal.
Performance preset
Please choose preset that fits your setup. This will adjust some settings for you.
Finished
You can always restart this wizard from the ALVR-tab on the left
